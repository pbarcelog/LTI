# Product Context – LTI App

## Overview
LTI is a recruitment management platform that allows companies to manage hiring processes from job creation to candidate selection.

## Users
- Recruiters
- Hiring managers

## Core Workflow
1. Create job
2. Publish job
3. Receive applications
4. Review applications
5. Run tests
6. Schedule interviews
7. Select candidates
8. Hire candidates

## Key Concepts
- Job
- Applicant
- Application
- Test
- Interview
- Hiring decision

## Scope
This phase focuses on creating the project structure (frontend, backend, database).
No business functionality is implemented yet.
